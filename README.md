# A_chatter · 进阶闲谈家

A_chatter 是 MaiBot 的日程、提醒、联网检索与主动发言插件。它负责任务创建、二次确认、权限管理、频率控制和上下文收集，并在合适的时机触发 Maisaka 主动发言。

作者：A-Dawn
主页：https://github.com/A-Dawn
许可证：GPL-3.0-or-later

## 功能概览

- **命令入口**：`/ac`、`/闲谈`、`/进阶闲谈`，三者等价
- **任务管理**：创建、确认、取消、查询、暂停、恢复、删除
- **二次确认**：所有任务创建后需在 5 分钟内确认；确认话术由 LLM 生成，支持自然回复或命令确认
- **日程调度**：支持单次（once）、间隔（interval）、Cron 表达式
- **硬提醒**：到点直接发送消息
- **主动发言**：注入上下文后触发 Maisaka 发言；若 Maisaka 无响应，由 LLM 生成兜底消息
- **联网检索**：通过 Tavily API 获取实时信息并压缩为事实摘要
- **自动发起**：根据目标扫描、频率函数和 LLM 判断，自动触发对话

## 安装

将本仓库放入 MaiBot 的插件目录：

```
MaiBot/
  plugins/
    A_chatter/
      _manifest.json
      plugin.py
      a_chatter/
```

依赖已在 `_manifest.json` 中声明。本地开发环境执行：

```bash
uv sync
```

## 配置

插件通过 `AChatterConfig` 进行配置，主要分组：

| 分组 | 说明 |
|------|------|
| plugin | 基础开关与版本 |
| permissions | 角色与跨聊天流权限 |
| whitelist | 用户、群聊、私聊白名单 |
| frequency | 自动发起频率参数 |
| targets | 单个目标的自动发起配置 |
| tavily | 联网检索 API Key |
| confirmation | 确认有效期与待确认上限 |
| proactive | 主动触发与兜底参数 |
| scheduler | 后台调度参数 |

> Tavily 联网检索需要填写 API Key。未配置时不影响其他功能，在未配置时执行联网检索会明确提示检索未执行。

完整配置说明见 [`docs/CONFIGURATION.md`](docs/CONFIGURATION.md)。

## 使用示例

**创建提醒：**

```
用户：/ac 新增 明天晚上八点提醒我交报告
Bot ：我先把这个提醒整理成草稿，请你确认……
用户：就这样，帮我设上
Bot ：已创建任务 ✓
```

**创建主动发言：**

```
用户：/闲谈 新增 明天上午九点问问大家项目进度
Bot ：这个会作为日程主动发言任务，请确认……
用户：确认
```

到点后，插件会收集最近聊天记录、记忆和可选的联网事实，注入上下文后触发 Maisaka 主动发言。

**创建联网摘要：**

```
用户：/ac 新增 每天早上八点整理 AI 新闻摘要
Bot ：我整理成了联网摘要草稿，请确认……
用户：/ac 确认
```

确认草稿时可以直接回复“确认”“可以”“就这样”“帮我设上”，也可以使用 `/ac 确认 [草稿ID]`。取消时可以回复“取消”“算了”“不要了”，或使用 `/ac 取消 [草稿ID]`。

## 权限说明


- 普通用户可以为自己的私聊创建个人任务
- 群聊任务需要配置允许或由超级管理员创建
- 跨聊天流创建默认关闭，需配置开放或具备超级管理员身份

目标格式示例：`qq:private:10000`、`qq:group:123456`
